# m87 CLI Examples

This directory contains examples and guides for using the m87 CLI.

## Structure

- **[features/](./features/)** - Documentation for core m87 features
  - Learn how to use specific CLI commands and capabilities
  - Reference documentation with working examples

- **[use-cases/](./use-cases/)** - Real-world use case tutorials
  - Step-by-step guides for accomplishing specific tasks
  - Practical recipes combining multiple features

## Getting Started

1. Start with [features/](./features/) to learn the basics
2. Explore [use-cases/](./use-cases/) for practical applications
